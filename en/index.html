Applied Cases

Representative cases

Examples where shipyard and manufacturing scan data became inspection, recognition, and automation outputs.

2025

Deep-learning spatial recognition automation module for container ship cargo holds

Structures and working spaces are recognized from cargo-hold point clouds, then object segmentation, spatial relationships, and inspection criteria are organized into automation-ready data.

  • Input: cargo-hold 3D point clouds
  • Analysis: AI object recognition, spatial reasoning, accessibility
  • Output: inspection and work criteria data package
2020-2021

3D point-cloud object detection for LNGC cargo holds

Deep-learning point-cloud processing and post-processing were applied to detect and separate scaffolding systems during cargo-hold measurement.

  • Input: LNGC cargo-hold scan data
  • Analysis: object detection, semantic segmentation, post-processing
  • Output: cleaned measurement target data
2016-2021

3D scanning and deviation analysis for small vessels and FRP molds

Scan data from small vessels and production molds was compared against reference geometry to validate dimensional control and process improvement opportunities.

  • Input: small vessel and FRP mold scan data
  • Analysis: registration, deviation map, quality criteria
  • Output: mold quality and improvement evidence
2025-2026

Collaborative robot welding automation and point-cloud surface extraction

Point clouds from constrained ship-block environments are converted into robot paths, collision checks, and welding automation input data.

  • Input: ship-block workspace point clouds
  • Analysis: surface extraction, path candidates, collision review
  • Output: robot simulation input data
