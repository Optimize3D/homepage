Applied Cases

대표 적용사례

선박·제조 현장의 스캔 데이터가 실제 검사, 인식, 자동화 데이터로 이어진 사례입니다.

2025

딥러닝 기반 컨테이너선 카고홀드 공간인식 자동화 모듈

조선소 컨테이너선 카고홀드 포인트클라우드에서 구조물과 작업공간을 인식하고, 객체 분할·공간 관계·검사 기준 데이터를 자동화 모듈로 정리합니다.

  • 입력카고홀드 3D 포인트클라우드
  • 분석AI 객체인식, 공간추론, 접근성 판단
  • 결과검사/작업 기준 데이터 패키지
2020-2021

LNGC 화물창 3D 포인트클라우드 객체 탐지

조선소 LNGC 화물창 편평도 계측 과정에서 스캐폴딩 시스템을 탐지·분할하기 위한 딥러닝 기반 포인트클라우드 처리와 후처리 연구를 수행했습니다.

  • 입력LNGC 화물창 스캔 데이터
  • 분석객체 탐지, 의미론적 분할, 후처리
  • 결과계측 대상 정리 및 검사 효율화
2026

3차원 포인트 클라우드 기반 객체 탐지와 공간 데이터 후처리 알고리즘을 활용한 선박 블록 지지대 설치 정확도 검사

실제 조선소에서 16개 지점 LiDAR 스캔으로 확보한 선박 블록 하부 포인트클라우드를 의미론적으로 분할하고, 개별 지지대의 중심과 높이를 추출해 기준 배치 대비 설치 편차를 정량 검사했습니다.

  • 입력선박 블록 하부 16개 지점 3D LiDAR 스캔
  • 분석PointVector 의미론적 분할, DBSCAN 군집화, RANSAC 형상 추정
  • 결과개별 지지대 중심·높이 추출 및 설치 편차 정량화
2025

위상-기하 특징 증강 조선소 환경 데이터셋 경량화

대용량 선박 블록·조선소 포인트클라우드에서 위상적으로 중요한 구조와 기하학적 국부 특징을 추출해 데이터 규모를 줄이고, 딥러닝 객체탐지에 필요한 형상 정보와 성능을 유지하는 경량 데이터셋을 구성했습니다.

  • 입력대용량 선박 블록·조선소 환경 3D 포인트클라우드
  • 분석TDA 지속적 호몰로지, FPFH 기하 특징 추출·특징 증강
  • 결과형상 품질과 객체탐지 성능을 유지하는 경량 학습 데이터셋
2016-2021

소형선박·FRP 몰드 3D 스캐닝 및 형상 편차분석

소형선박과 양산몰드의 스캔 데이터를 기준 형상과 비교해 정도관리 및 공정개선 가능성을 검증했습니다.

  • 입력소형선박, FRP 몰드 스캔 데이터
  • 분석정합, 편차맵, 품질 기준 검토
  • 결과몰드 품질관리 및 개선 근거
2025-2026

포인트클라우드 기반 피지컬AI·협동로봇 작업성 검토

선박블록 포인트클라우드 환경, 3D 비전 센서, 3D 프린팅 시험편을 연결해 로봇 경로·충돌·접근성·용접선 추출 가능성을 검토합니다.

  • 입력선박블록 포인트클라우드, 로봇 셀, 3D 비전 센서 조건
  • 분석공간추론, 충돌 감지, 용접선·표면 추출
  • 결과피지컬AI 실험 셋업 및 로봇 실행 데이터

Operating Model

3D 스캔 이후 병목을 줄입니다.

단순 3D 스캐닝 결과 확인에서 벗어나, 스캔 데이터를 인공지능으로 지능화해 검사·역설계·품질판정·피지컬AI 실행 데이터까지 확장할 수 있는 기반을 만듭니다.

01

Field Capture

포인트클라우드, CAD, 사진, 검사 기준과 현장 제약을 확인합니다.

02

Reference Model

좌표계, 기준 형상, 허용 오차와 판정 항목을 정의합니다.

03

AI & Metrology

편차맵, 특징 추출, 객체 분할, 공간추론과 리포트 형식을 검증합니다.

04

Execution Package

검사 리포트, CAD 재구성, 지그 설계와 피지컬AI 실행 데이터로 납품합니다.
